# client-onboarding (paid)

Walks a new client through the operator's standard kickoff in a single
async session. Collects access tokens, brand assets, and goals, then
files them into the operator's client folder structure.

## Tier
Paid — included in every paid bundle.

## Triggers
The agent invokes this skill on the first session with a new client, or
when the operator runs a recurring re-onboarding for an account.

## Inputs
- A new client name.
- The operator's onboarding template — questions, required artefacts.
- A target folder root in the operator's client tree.

## Outputs
- A populated client folder under the operator's client root.
- A welcome packet sent through whichever channel the operator has wired
  up.

## Compatibility
Hermes default; OpenClaw fork; BYOA Codex; BYOA Claude Code.
