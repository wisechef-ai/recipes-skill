# cold-outreach (paid)

Drafts and schedules cold outreach sequences for an agency targeting
small businesses. Pulls a prospect list from the operator's CRM, clusters
prospects by vertical, drafts a four-touch sequence per cluster, and
schedules sends through whichever provider the operator has wired up.

## Tier
Paid — included in every paid bundle.

## Triggers
The agent invokes this skill when the operator asks to draft outreach,
warm a list of leads, or run a multi-touch sequence over a defined
window.

## Inputs
- A prospect list — CSV with one row per lead.
- A campaign brief — vertical, value proposition, call to action.
- A send window — days of week, send rate per day.

## Outputs
- A drafted sequence per cluster, queued in the connected provider.
- A daily summary of opens, replies, and bounces, written to the
  operator's notes directory.

## Compatibility
Hermes default; OpenClaw fork; BYOA Codex; BYOA Claude Code.
