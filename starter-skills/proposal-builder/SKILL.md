# proposal-builder (paid)

Drafts a client proposal from a discovery brief. Pulls the operator's
boilerplate, fills in scope, timeline, and pricing tiers, and writes a
PDF plus an editable source document.

## Tier
Paid — included in every paid bundle.

## Triggers
The agent invokes this skill whenever the operator asks for a proposal,
a statement of work, or a quote.

## Inputs
- A discovery brief — outcome, constraints, budget range.
- The operator's boilerplate profile — tone, terms, default warranties.
- A pricing model — fixed, hourly, or tiered.

## Outputs
- A PDF proposal in the operator's proposals directory.
- An editable source draft for last-mile review.

## Compatibility
Hermes default; OpenClaw fork; BYOA Codex; BYOA Claude Code.
